#  Clasificación de Residuos con Deep Learning

##  Descripción del Proyecto
Este proyecto utiliza **Transfer Learning (Aprendizaje por Transferencia)** y **Fine-Tuning** para clasificar imágenes de residuos en diferentes categorías (Orgánico y Reciclable). El objetivo es optimizar la separación de basura mediante visión artificial.

##  Metodología
* **Dataset:** Imágenes de residuos clasificados.
* **Modelo Base:** Uso de una red neuronal pre-entrenada (Transfer Learning).
* **Fine-Tuning:** Ajuste de las últimas capas para especializar el modelo en el dataset de residuos.
* **Librerías:** TensorFlow, Keras, Matplotlib, NumPy.
